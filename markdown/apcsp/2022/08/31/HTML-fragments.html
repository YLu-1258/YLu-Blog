HTML Fragments and Markdown

HTML fragments are just a part of the total HTML source on our site.

Table Fragments Fastpages allows to build tables in HTML or Markdown. Everything on the site is built in HTML, the markdown is converted into individual HTML fragments which are assembled to form webpage. Tables that we create and use in markdown are also read, interpreted and turned into html tables.

tl;dr EVERYTHING ON THE WEBPAGE IS IN HTML

images “” element in HTML allows us to specify cetain image parameters, more flexible than markdown method.

links “” element with href in HTML
